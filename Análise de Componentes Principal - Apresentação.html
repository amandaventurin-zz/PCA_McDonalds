INTRODUÇÃO¶

O que é o PCA?¶

A Análise de Componentes Principais (ACP) ou Principal Component Analysis (PCA) é um procedimento matemático que utiliza uma transformação ortogonal (ortogonalização de vetores) para converter um conjunto de observações de variáveis possivelmente correlacionadas num conjunto de valores de variáveis linearmente não correlacionadas chamadas de componentes principais.

Pra que serve o PCA?¶

Ao aplicar esse procedimento, é possível reduzir o número de variáveis a somente aquelas que tem grande importância naquele conjunto de dados. Isso é um metodo muito usado para análise e tratamento dos dados, pois com esse subconjunto é possível chegar às mesmas conclusões que com o conjunto inteiro mas com menos esforço. O PCA tem sido muito utilizado nas criações de modelos de predições em Machine Learning (ML), principalmente porque diminui o custo do processamento dos dados.
