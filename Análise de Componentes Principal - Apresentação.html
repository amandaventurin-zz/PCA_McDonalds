INTRODUÇÃO¶

O que é o PCA?¶

A Anlise de Componentes Principais (ACP) ou Principal Component Analysis (PCA) um procedimento matemtico que utiliza uma transformao ortogonal (ortogonalizao de vetores) para converter um conjunto de observaes de variveis possivelmente correlacionadas num conjunto de valores de variveis linearmente no correlacionadas chamadas de componentes principais.

Pra que serve o PCA?¶

Ao aplicar esse procedimento, possvel reduzir o nmero de variveis a somente aquelas que tem grande importncia naquele conjunto de dados. Isso um metodo muito usado para anlise e tratamento dos dados, pois com esse subconjunto possvel chegar s mesmas concluses que com o conjunto inteiro mas com menos esforo. O PCA tem sido muito utilizado nas criaes de modelos de predies em Machine Learning (ML), principalmente porque diminui o custo do processamento dos dados.
